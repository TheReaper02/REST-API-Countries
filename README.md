**REST Countries API with Theme Switcher**

Built a responsive React app that integrates with the REST Countries API. Users can:

- Browse all countries in a searchable, filterable grid.
- Search for countries by name and filter by region.
- View detailed information for each country, including border countries.
- Toggle between light and dark mode for the interface.
- Enjoy a modern, accessible UI styled with Tailwind CSS and React Router for navigation.
